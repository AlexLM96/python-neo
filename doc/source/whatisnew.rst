*****************
What is new ?
*****************

New features compared to neo1:
 * new schema more consistent.
 * new objects: RecordingChannelGroup, EventArray, AnalogSignalArray, EpochArray
 * Neuron is now Unit
 * use the python Quantity module for almost everything that can use a units.
 * Some object directly hinerits Quantity : SpikeTrain, AnalogSignal, AnalogSIgnalArray instead of having an attribute for data.
 * Attributes are classifyed in 3 categories: necessary, recommended, free.
 * lazy and cascade keywork are added to all IOs
 * python 3 support
 * better tests
